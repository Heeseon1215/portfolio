OVERVIEW

평화와 사랑 그리고  벤앤제리스 

벤앤제리스는 아이스크림을 통해 세상을 더행복하게 만드는 데 의미를 부여하고 있습니다.
이들은 평화와 사랑을 전파하고, 그러다 보면 아이스크림을 통해 세상을 바꿀 수 있다고 믿습니다.
벤앤제리스는 맛있는 아이스크림을 넘어, 사회적 책임을 다하며 세상을 바꾸는 힘을 믿습니다.

PROJECT GOAL

벤앤제리스 모바일 디자인

브랜드 스타일 유지

벤앤제리스의 독특하고 유쾌한 일러스트 스타일을 강조하여 브랜드의 개성과 활기를 잘 담아낸 모바일 페이지 디자인을 목표로 합니다.

React 모바일 사이트

React의 프레임워크를 활용하여 밴엔제리스의 모바일 사이트를 디자인하고 콘텐츠를 손쉽게 추가 및 수정할 수 있는 시스템을 구축하는 것을 목표로 합니다.

학습 내용 활용

GSAP 스크롤 애니메이션 및 CSS 애니메이션, SCSS 등 지금까지 학습한 다양한 기술들을 종합적으로 적용한 웹사이트를 제작하는 것을 목표로 합니다.

PROBLEM & SOLUTION

01
GSAP 스크롤 애니메이션이 왜 실행이 안될까?

GSAP를 활용한 스크롤 애니메이션이 의도되로 실행이 안되며 이미지는 멈춰있어 스크롤에 따른 움직임이 발생하지 않은 상황 발생

scss 모듈을 사용한 클래스 선택자로 인한 오류. 따라서 useRef를 활용하여 DOM요소를 직접 참조 후 JavaScript로 트리거와 위치를 지정하여 정상적으로 작동하게 설정

02
메뉴의 토글버튼과 스크롤은 어떻게 해야되는거지?

기존 jQuery로만 작업하던 메뉴 토글 버튼을 React 방식으로 작업을 하면서 JavaScript의 삼항 연산자로 작성해야하는 상황 발생

React의 useState를 활용하여 메뉴의 열림 닫힘 (isMenu)와 스크롤 상태를 관리하고 useEffect를 사용하여 스크롤 이벤트 감지하게 설정

RESULT

프로젝트 리뷰

  • 처음 작업한 React와 SCSS

    이번 프로젝트는 리액트를 활용한 첫 프로젝트로 SPA를 구현하는 도전적인 경험이였습니다. GSAP와 Swiper 적용, 클래스 선택자 문제 등 다양한 문제를 맞닥뜨렸으나 이를 해결하기 위해 스스로 공부하며 문제를 해결해나갔습니다. 그 과정을 통해 리액트에 대한 이해도를 높일 수 있었고, 앞으로 리액트에 적용할 다양한 스타일 프레임워크를 활용하여 어플리케이션을 개발하고, 이벤트 구현 기술을 지속적으로 학습해야겠다고 느꼈습니다.

  • 모바일 페이지 디자인

    타이포그래피를 활용한 유쾌한 디자인을 구현하며 타이포가 사이트의 분위기에 미치는 영향과 CSS 애니메이션을 통한 사용자 경험 개선 방법을 배울 수 있었습니다. 다양한 색감과 타이포를 중점으로 한 디자인 작업을 통해, 향후 웹사이트 제작 시 이를 효과적으로 적용하는 방법을 익힐 수 있었던 유익한 프로젝트였습니다.
